# Row Counter

A small counter app for tracking rows and rounds which is useful for anything done in repeated sets (knitting, workouts, reps of any kind). Increase or decrease the count, and save a snapshot of previous totals.

**Live:** https://row-counter-inky.vercel.app/

---

## Features

- Increase / decrease a running count
- Save previous counts for reference
- Simple, distraction-free interface

---

## Built with

- HTML
- CSS
- JavaScript

---

## Notes

Built as a front-end fundamentals project without frameworks or build step, just vanilla HTML/CSS/JS deployed as a static site.
